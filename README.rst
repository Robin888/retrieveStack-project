Install
-------------------------------------------------------------------------------

``retrieveStack`` is released on PyPI, so all you need is:

.. code-block:: console

	$ pip install retrieveStack

To upgrade to latest version:

.. code-block:: console

	$ pip install --upgrade retrieveStack
